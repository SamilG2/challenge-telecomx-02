# 🤖 Challenge TelecomX - Parte 02

## 📌 **Propósito del Proyecto**
Predecir el Churn (Cancelación) de clientes en base a variables relevantes.

## 📁 **Estructura del Proyecto**
El análisis de datos completo se encuentra en el notebook `TelecomX02_LATAM.ipynb`. La fuente de datos utilizada fue extraido de la primera parte del challenge, está disponible en el archivo `datos_tratados.csv` de este repositorio.
  
Se llevaron a cabo 4 pasos principales:
 1. **Extracción**
 2. **Preparación de los datos**
 3. **Correlación entre las variables**
 4. **Modelos Predictivos**
 5. **Conclusiones**

Finalmente, con base en estos criterios, se desarrolló un informe que detalla lo siguiente:
- Introducción
- Análisis Exploratorio y Preparación de datos
- Comparación del Rendimiento del Modelo
- Factores más influyentes en la Cancelación
- Estrategias de Retención Propuestas
- Conclusiones

## 📊 **Gráficos Utilizados**
- Mapa de calor (Heatmap)
- Gráfico de barras agrupado
- Gráfico de caja y bigotes

Estos gráficos proporcionan claridad visual sobre las comparativas necesarias para tomar la mejor decisión.

## 🔗 **Instrucciones para ejecutar el Notebook**

1. Descarga el archivo `TelecomX02_LATAM.ipynb` con el Dataset `datos_tratados.csv`.
2. Súbelo a [Google Colab](https://colab.research.google.com/).
3. Selecciona **"Ejecutar todo"** para reproducir el análisis completo.
